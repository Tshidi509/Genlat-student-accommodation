# Project Overview
This project is a student accommodation management system designed to connect students with accommodations that meet their needs. It provides an easy-to-use interface for students and accommodation providers.

## Features
- Search for available accommodations
- Filter results by criteria (price, location, type)
- Contact accommodation providers directly
- User authentication for students and providers
- Admin panel for managing listings

## Setup Instructions
1. Clone the repository: `git clone https://github.com/Tshidi509/genlat-student-accommodation.git`
2. Navigate to the project directory: `cd genlat-student-accommodation`
3. Install the required dependencies: `npm install`
4. Start the development server: `npm start`

## Technologies Used
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT
